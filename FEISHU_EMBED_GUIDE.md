# 如何创建飞书多维表格的干净嵌入链接

由于 iframe 跨域限制，CSS 无法直接隐藏飞书工具栏。最有效的解决方案是使用飞书的"发布到网络"功能。

## 步骤 1: 在飞书中创建公开链接

1. 打开你的飞书多维表格：
   https://my.feishu.cn/base/GrYvbphumaTN17skgUicG7KFnCf?table=tblGLAs5S4rZo59n&view=vew5xJ57BV

2. 点击右上角的 **"分享"** 按钮

3. 在分享弹窗中：
   - 开启 **"发布到网络"** 或 **"获取分享链接"**
   - 选择 **"任何人可查看"**
   - 复制生成的公开链接

4. 公开链接格式通常类似：
   ```
   https://feishu.cn/base/GrYvbphumaTN17skgUicG7KFnCf?from=pub
   ```

## 步骤 2: 更新环境变量

获取到新的公开链接后，请告诉我，我会帮你更新到：

1. **本地环境变量**: `.env.local`
2. **Vercel 生产环境**: `FEISHU_BASE_URL`

## 为什么公开链接更好？

公开链接的优点：
- ✅ 没有工具栏（分享、自动化等按钮）
- ✅ 专门为嵌入优化
- ✅ 加载速度更快
- ✅ 更干净的界面

## 安全性说明

即使使用公开链接，安全性仍然由我们的网站保护：
- ✅ 用户必须登录才能访问
- ✅ 用户必须有付费订阅
- ✅ URL 通过 API 代理获取，不直接暴露
- ✅ 飞书链接本身是公开的，但只有通过我们网站的验证用户才能访问
